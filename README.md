
# SnA's Personal Plugins for [`musikcube`](https://github.com/clangen/musikcube)

My collection of plugins for `musikcube` for my own twisted use cases that are borderline unique.

> [!IMPORTANT]
> Apart from vendored code, all code are generated by LLMs. While I have test-driven my code over actual music I listen, the code may contain structural errors I lack the domain depth to catch. C++ is not my strength, neither am I deeply knowledgeable about how different OSes treat its audio stack. It's **holistically working**, not provably correct. I'm sorry!
>
> All commits ***will*** list the precise LLM(s), harness(es) and operation context in `Generated-By: ` Git trailer and signed off by me per <https://openinfra.org/legal/ai-policy>


> [!NOTE]
> Any mention to commercial products here is not equivalent to my endorsement. These are my things, bought with my money, and are simply known-good cases I tested against.

## Content

1. WASAPI exclusive mode (EM) + `libsoxr`resampling.
    - I have tested this only with my DAC (FiiO K11 R2R, firmware `M79`) which tops out at PCM 32/384. 
2.  WASAPI EM tester.
    - You have [Eugene Muzychenko's VAC](https://vac.muzychenko.net/en/) to thank for this.

## Building
Please follow [upstream's instruction](https://github.com/clangen/musikcube/wiki/building) for tools needed (Visual Studio 2022 / MSBuild). You shouldn't need any additional dependencies.
```sh
$ git clone --recursive https://github.com/ShinonomeNoAlice/sna-mcube
$ cd sna-mcube
$ msbuild musikcube-plugins.sln /p:Configuration=Release /p:Platform=x64
```
The compiled output will be generated under the `build/` directory:
- WASAPI Exclusive Mode Plugin: `build\bin\x64\Release\plugins\wasapiexclusiveout.dll`
- SoX Resampler Shared Library: `build\bin\x64\Release\soxr.dll`
- WASAPI Exclusive Mode Tester: `build\bin\x64\Release\tools\test_wasapi.exe`

Then copy/symlink:
1. `wasapiexclusiveout.dll` into your `musikcube`'s `plugins/` folder.
2. `soxr.dll` into your `musikcube` root folder (where `musikcube.exe` is located).
